# 🎨 Pixel Art Canvas

Un créateur de pixel art interactif avec sélection de couleurs et fonctions de dessin.

![Capture d'écran](screenshot.png)

## ✨ Fonctionnalités
- Grille 10x10 cliquable
- Palette de couleurs personnalisable
- Boutons :
  - 🎲 Random (couleur aléatoire)
  - 🧹 Clear (effacer tout)
- Design kawaii avec animations

## 🛠 Technologies
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript Vanilla

## 🚀 Comment l'utiliser
1. Cliquez sur une cellule pour la colorer
2. Choisissez une couleur avec le sélecteur
3. Utilisez Shift+Click pour effacer
